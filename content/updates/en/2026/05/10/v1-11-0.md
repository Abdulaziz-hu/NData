## v1.11.0 - 2026-05-10

### New Features
- Bookmarks: save items locally, persists across sessions
- Compare: side-by-side spec comparison for up to 4 items
- Sort by price (asc/desc) and date added (new/old)
- Card image previews with halftone shader effect
- Auto image scroll on card hover (multi-image items)
- Copy Specs button on each card
- Keyboard shortcut: press / to focus search bar
- Multi-language support: English, Arabic, Japanese, German
- Translation files under /lang/
- EUR and GBP currencies added
- New categories: gaming, monitors, networking
- Auto ID assignment (prefix-NNNNNN format)
- Full API docs page at /api/docs
- Live API explorer at /api/api with filter, sort, download
- Bookmark button in product modal header
- Modal shows bookmark state in sync with card

### Fixes
- Theme toggle icon no longer disappears after re-render
- Burger menu icon stays consistent across state changes
- All icon elements use data-theme-icon attribute for reliable updates
- simple-func.js refactored to use data-theme-icon pattern

### Content
- Removed all PLACEHOLDER.txt files
- Added first real news, blog, and updates content files
- Added /lang/en.json, ar.json, jp.json, de.json
